# Project: Demo Repository

Vite React repository used as a testing ground for the A11y Context Project. 

## Background: A11y Context Project

The purpose of the A11y Context Project is to make accessibility best practices available to AI coding assistants in order to improve their output. It consists of an AI assistant layer (router and rules), an MCP as connective tissue, and a repository of pattern documentation.

This repository is used to test MCP commands and code output.
